1.0 Root Finding

Root finding algorithms are essential for solving equations where one needs to find the value of x for which f(x) = 0. Below we discuss several popular methods used in numerical analysis.

1.1 Bisection Method

The bisection method is a simple, reliable, and robust method for root finding. It works by repeatedly dividing the interval in half and selecting the subinterval in which the function changes sign, and therefore must contain a root.
